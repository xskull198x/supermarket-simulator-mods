Throw Restock

Version 1.3.7 for Supermarket Simulator with BepInEx IL2CPP.

Throw an open product box at a shelf to stock matching items. If the hit shelf has the wrong label, the mod tries to stock the closest matching tagged shelf instead. Full shelves leave leftovers in the box. Empty boxes can be thrown at trash cans to dispose of them.

Download ThrowRestock v1.3.7

Install

  1. Install BepInEx IL2CPP for Supermarket Simulator.
  2. Extract this zip directly into your Supermarket Simulator game folder.
  3. The zip already contains BepInEx/plugins/ThrowRestock/ThrowRestock.dll.
  4. For multiplayer, the host and guests should install the mod locally.
